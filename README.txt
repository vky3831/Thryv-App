Thryv - Single-file static webapp

How to use:
1. Open index.html in a modern browser (Chrome/Edge/Firefox).
2. On first load create a profile by entering Name and Passkey.
3. After creating profile you'll see the app list and can open embedded sample apps.
4. Logout will hide the app; closing the tab automatically clears the session (sessionStorage).
5. Data (profile) is stored locally in IndexedDB under database 'ThryvDB' object store 'profile'.

Files:
- index.html
- style.css
- script.js
- apps/sample1/index.html
- apps/sample2/index.html
